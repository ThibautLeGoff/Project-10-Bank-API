Client minimal React (Vite) convertion des fichiers dans `designs/`.

Quick start:

1. Copier les images depuis `designs/img` vers `client/public/img` (créez le dossier `public/img` si nécessaire).
2. cd client
3. npm install
4. npm run dev

Notes:
- Ce client est volontairement minimal: routing hash, pas d'API intégrée.
- Pour la police/iconographie utilisez la CDN Font Awesome (inclus via class names).
